# AI Prompt Sharing & Marketplace Platform (Client)

## Purpose

AI Prompt Sharing & Marketplace Platform is a modern web application where users can discover, create, bookmark, review, and manage AI prompts for various AI tools such as ChatGPT, Gemini, Claude, and Midjourney. The platform also includes premium content, role-based dashboards, and secure authentication.

## Live URL

https://ai-prompt-client-itsasmaulhosna-maya1234.vercel.app

## Key Features

- JWT Authentication
- Google Login
- Role-Based Access Control (User, Creator, Admin)
- Featured & Trending Prompts
- Backend Search, Filter & Sorting
- Prompt Bookmark System
- Copy Prompt Functionality
- Review & Rating System
- Prompt Reporting
- Stripe Premium Subscription
- User Dashboard
- Creator Dashboard
- Admin Dashboard
- Analytics Charts
- Responsive Design
- Framer Motion Animations
